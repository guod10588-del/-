# 📖 技术笔记

## 📂 目录说明

### 编程语言/

- **Python** - 数据处理、Web 框架、自动化
- **JavaScript** - 前端、Node.js、框架
- **TypeScript** - 类型系统、高级特性
- **Go** - 并发、微服务、工具开发
- **Rust** - 内存安全、性能优化
- **SQL** - 数据库、查询优化

### 框架工具/
常用框架和开发工具
- **前端框架** - React、Vue、Angular
- **后端框架** - Django、FastAPI、Spring
- **构建工具** - Webpack、Vite、Gradle
- **测试框架** - Jest、pytest、unittest
- **开发工具** - Git、Docker、VS Code

### 运维部署/

- **CI/CD** - GitHub Actions、GitLab CI
- **监控** - Prometheus、ELK、Grafana
- **云服务** - AWS、Azure、阿里云
- **Linux** - 系统管理、性能调优

### 系统设计/
架构设计和最佳实践
- **架构模式** - MVC、微服务、事件驱动
- **设计模式** - 单例、工厂、观察者等

- **API 设计** - RESTful、GraphQL、gRPC
- **安全** - 认证、授权、加密

## 🏷️ 推荐标签

- `高级` - 高级特性
