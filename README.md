# 个人知识库 📚

这是个人技术知识库，用于系统地整理和储存各类技术笔记、文档、资源和最佳实践。

## 📁 目录结构

```
├── 笔记/                    # 技术笔记和学习心得
│   ├── 编程语言/            # Python、JavaScript、Go 等
│   ├── 框架工具/            # 框架、库和开发工具
│   ├── 运维部署/            # DevOps、容器、CI/CD
│   └── 系统设计/            # 架构、性能、最佳实践
├── 提示词/                  # AI 提示词集合
│   ├── 编程类/              # 代码审查、调试、重构
│   ├── 写作类/              # 文档、博客、评论
│   └── 效率类/              # 学习、规划、管理
├── 资源/                    # 推荐资源和参考链接
├── 模板/                    # 项目模板和配置文件
└── 代码片段/                # 常用代码和工具函数
```

## 🎯 快速开始

### 查看笔记
进入各目录查看 README 文件了解详细内容

### 添加内容
1. 选择合适的目录
2. 参考 [贡献指南](./贡献指南.md)
3. 创建 Markdown 文件
4. 提交更改

### 搜索笔记
使用 GitHub 搜索功能或在本地使用 `grep`：
```bash
grep -r "关键词" .
```
