# 🤝 贡献指南

感谢你有兴趣为这个知识库做贡献！本指南将帮助你了解如何参与。

## 📝 什么样的内容是欢迎的？

✅ **欢迎的内容**：
- 新的技术笔记和学习心得
- 实用的代码片段和最佳实践
- 有效的工具和资源链接
- 问题的解决方案
- 改进现有文档
- 修复错误和不准确的地方

❌ **不欢迎的内容**：
- 广告和推广
- 复制的内容（需要原创或署名）
- 过时的信息（未标注）
- 明显的垃圾内容

## 🎯 贡献方式

### 1. 添加新笔记

#### 步骤
1. 选择合适的目录（笔记/、提示词/等）
2. 创建 Markdown 文件
3. 按格式编写内容
4. 提交 Pull Request

#### 文件命名
- 使用中文或英文（保持一致）
- 单词间使用 `-` 分隔
- 清晰表达文件内容

**示例**：
```
笔记/编程语言/python-列表推导式.md
笔记/运维部署/docker-常用命令.md
代码片段/javascript/数组-过滤重复元素.js
```

```markdown

### 3. 编辑现有内容

#### 修改类型
- 📝 **改进** - 优化表达、补充���容
- 🐛 **修复** - 纠正错误、过时信息
- 🔍 **澄清** - 提高清晰度
- ✨ **增强** - 添加示例、相关链接

#### 修改步骤
1. 编辑文件
2. 在文件开头更新日期
3. 在末尾的更新日志中记录改动
4. 提交 Pull Request

### 4. 提交流程

#### 方式 1: 通过 GitHub 网页

1. 进入仓库相应目录
2. 点击 "Add file" → "Create new file"
3. 输入文件名和内容
4. 点击 "Propose new file"
5. 创建 Pull Request

#### 方式 2: 通过 Git CLI

```bash
# 1. Fork 仓库
git clone https://github.com/your-username/-.git
cd -

# 2. 创建新分支
git checkout -b add-python-notes

# 3. 添加你的内容
mkdir -p 笔记/编程语言
echo "# Python 笔记" > 笔记/编程语言/python-基础.md

# 4. 提交更改
git add .
git commit -m "docs: add Python basic notes"

# 5. 推送到你的 Fork
git push origin add-python-notes

# 6. 在 GitHub 上创建 Pull Request
```

## ✅ 检查清单

提交前请确认：

- [ ] 内容在正确的目录
- [ ] 文件名清晰有意义
- [ ] Markdown 格式正确
  - [ ] 标题层级合理（# > ## > ###）
  - [ ] 代码块有语言标记
  - [ ] 链接格式正确
- [ ] 包含必要的元信息
  - [ ] 创建日期
  - [ ] 标签
  - [ ] 简短摘要
- [ ] 内容原创或已署名
- [ ] 相关链接已添加
- [ ] 拼写和语法检查
- [ ] Pull Request 描述清楚

## 🏷️ 推荐标签

使用这些标签来分类你的内容：

### 主题标签
- `编程语言` - 编程语言相关
- `框架工具` - 框架和工具
- `运维部署` - DevOps 相关

### 内容类型

- `高级` - 高级特性
- `最佳实践` - 推荐做法
- `教程` - 学习教程
