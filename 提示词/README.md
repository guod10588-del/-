# 🎯 AI 提示词库

精心收集和优化的 AI 提示词集合，用于高效完成各类任务。

## 📂 分类导航

### 编程类/
与代码相关的提示词
- **代码审查** - 高效审查代码质量
- **调试建议** - 快速定位和解决 Bug
- **代码重构** - 改进代码结构和可读性
- **性能优化** - 性能分析和优化建议
- **文档生成** - 自动生成代码文档
- **测试编写** - 生成测试用例

### 写作类/
文字写作相关的提示词
- **技术文档** - 撰写清晰的技术文档
- **博客文章** - 创意博客内容
- **代码注释** - 生成高质量注释
- **邮件沟通** - 专业沟通邮件
- **演讲稿** - 技术演讲内容

### 效率类/
提升工作效率的提示词
- **学习计划** - 制定学习路线
- **任务管理** - 任务分解和优先级
- **时间管理** - 时间规划建议
- **问题分析** - 系统性分析问题
- **决策支持** - 帮助做出决策

### 创意类/
创意工作和头脑风暴
- **产品设计** - 功能设计建议
- **营销文案** - 创意营销内容
- **头脑风暴** - 生成创意想法
- **用户体验** - UX 设计建议

## 💡 如何使用提示词

### 基本步骤

1. **选择合适的提示词**
   - 根据任务选择相关分类
   - 找到最接近的模板

2. **个性化定制**
   - 根据具体情境修改参数
   - 添加你的特定背景信息
   - 调整期望输出的格式

3. **迭代优化**
   - 根据初始结果反馈
   - 逐步改进提示词
   - 保存优化后的版本

### 提示词模板结构

```
# [任务名称]

**目的**: 简述这个提示词的用途
**适用场景**: 什么情况下使用
**模型推荐**: 建议的 AI 模型

## 提示词

[完整的提示词内容]

## 使用示例

**输入**:
```
你的具体信息
```

**输出示例**:
```
预期的输出结果
```

## 优化技巧

- 修改点 1: ...
- 修改点 2: ...

## 相关提示词
- [其他相关提示词]()
```

## 🔧 优化技巧

### 1. 明确上下文
```
"你是一个有 10 年经验的 Python 工程师"
```

### 2. 指定输出格式
```
"请用 Markdown 格式回答，包括代码示例"
```

### 3. 提供示例
```
"示例输入：...\n示例输出：..."
```

### 4. 分步骤
```
"请按以下步骤完成：\n1. ...\n2. ...\n3. ..."
```

### 5. 添加约束
```
"回答长度不超过 200 字"
"只提供最重要的 3 个建议"
```

## 📊 提示词评估

评估提示词效果的标准：
- ✅ **相关性** - 输出是否符合需求
- ✅ **准确性** - 信息是否正确
- ✅ **实用性** - 是否可直接应用
- ✅ **清晰性** - 是否易于理解
- ✅ **一致性** - 结果是否稳定
